## 9. Archives — case study template (`/archives/non5`, `/archives/royal-exhibition-building`, `/archives/aster-bar-x-masterchef-david-tan`)

Template for individual event/activation write-ups. All three archive entries are now live (the latter two were unpublished at the time of the initial scrape and were added in a follow-up pass once the client published them).

**Shared shape**: full-bleed photo hero with centered title + date overlay → tag row → second full-bleed photo → "In collaboration with [partner]" → 2–3 body paragraphs → a supporting image block (sponsor logos or extra photos, varies per page) → related-cards grid (all 3 archive entries, including a self-link back to the current page — this self-referential behaviour is also how the live Framer CMS collection renders it) → footer.
